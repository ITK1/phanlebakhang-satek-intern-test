# Trang Quản Lý Công Việc Cá Nhân - SATEK Intern Test

Dự án được xây dựng như một phần của bài kiểm tra năng lực vị trí Thực tập sinh React tại **SATEK**.

## 🚀 Công nghệ sử dụng
- **React (Vite):** Tối ưu tốc độ phát triển và build.
- **Tailwind CSS v4:** Sử dụng engine mới nhất để xây dựng giao diện nhanh, hiện đại và Responsive.
- **LocalStorage:** Lưu trữ dữ liệu trực tiếp tại trình duyệt người dùng.

## ✨ Tính năng chính
- [x] **CRUD:** Thêm, sửa trạng thái và xóa công việc.
- [x] **Phân loại:** Quản lý công việc qua 3 cột (Cần làm, Đang làm, Hoàn thành).
- [x] **Tìm kiếm & Lọc:** Tìm kiếm nhanh theo tên công việc.
- [x] **Thống kê:** Theo dõi tổng số task, task hoàn thành và task quá hạn.
- [x] **Cảnh báo Deadline:** Hiển thị màu sắc cảnh báo cho các công việc sắp đến hạn hoặc đã quá hạn.
- [x] **Responsive:** Giao diện hiển thị tốt trên cả Desktop và Mobile.
